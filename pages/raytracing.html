We broke Ray Tracing up into parts:

We had a scene class which contained our main method. We would parse command line input for:
  • Objects : Sphere or Triangle, material, color, size and location and possible transformations
  • Lights - Type (diffuse, point, ambient), color, location
  • Camera - 3D vector location of where the camera eye was in the scene.
  • OBJ parser - Parses OBJ files for triangle objects.

Then with the list of primitive objects (triangles and/or spheres) and lights we call scene.render()
Scene.render():

  • Sets up the raytracer with the camera(Eye) location, image plane Coordinates and list of primitive objects and lights.
  • Iterates though each pixel in the image plane and and creates samples(x, y).
  • Raytracer.trace() calculates the color, shadow, reflection, and possible transformation from a ray sent from the Eye location though the center of each sample.
  • Stores that sample and corresponding color in a film class.
  • Writes Image into a ppm file in film class.
